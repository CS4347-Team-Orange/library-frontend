# Library Frontend


## Project setup
```
yarn install
```

## Run in container (production mode)
```
bash start.sh
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```

See [Configuration Reference](https://cli.vuejs.org/config/).
